# DynamicPanelModels.jl

Documentation for [DynamicPanelModels.jl](https://github.com/MichalS16/DynamicPanelModels),
a Julia package for estimating dynamic panel data models (Arellano-Bond Difference GMM,
Blundell-Bond System GMM, and Anderson-Hsiao IV) with robust and Windmeijer-corrected
standard errors, GMM diagnostics, and diagnostic plotting.

See the [README](https://github.com/MichalS16/DynamicPanelModels#readme)
for a quick-start example, and the [API Reference](@ref) for full function documentation.
