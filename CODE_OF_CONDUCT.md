# Code of Conduct

This project follows a simple standard: be respectful and constructive.

- Be welcoming to newcomers and considerate of different experience levels.
- Critique code and ideas, not people. Disagreements about econometric
  formulas or implementation choices are normal — keep them technical.
- No harassment, discriminatory language, or personal attacks, in issues,
  pull requests, or any other project space.
- Assume good faith; ask questions before assuming bad intent.

Reports of unacceptable behavior should be made privately to the maintainer
(see the GitHub profile for contact details), not in a public issue.
Maintainers may edit, remove, or reject contributions that don't align with
this standard.

This is a short, project-specific summary; for a fuller reference see the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
